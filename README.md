Eduxcate — Student Management System

Eduxcate is a full-stack Student Management System developed using Spring Boot, MySQL, HTML, CSS, and JavaScript. 
This project helps manage student records efficiently through a modern dashboard interface with secure login authentication and real-time database integration.

Features

* Secure Login Authentication
* Add Student Records
* View Student Details
* Delete Student Records
* Dashboard Overview
* Student Statistics
* Responsive Modern UI
* REST API Integration
* MySQL Database Connectivity
